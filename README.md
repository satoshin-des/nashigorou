# なしごろうの比率合わせゲーム

## これは何？
りん〇ろうではなく，なしごろうというキャラクターの大きさをマウスポインタで自由に動かして，見本のなしごろうの縦横比にどれだけ近づけられたかを競うゲームのソースです．

## 遊び方
マウスポインターでなしごろうを動かして比率を見本に合わせます．
遊んでればなんとなく遊び方が分かっていく筈です．
